# Todo List Website

## [Visit Website](https://todo-list-iohl.onrender.com)

> CRUD with fastapi practice #2

Todo List website for people who often forget their todo thing

## 🛠️ Stacks

- Svelte By Vite (Frontend)
- Fastapi Python (Backend)
- Supabase (DataBase)

## ✨ Features

- Can upload your todo things
- Control if its done or not by bool type
- Mobile React website to use well and easily
- Show modal when you try to delete the todo list

## 🚀 Execute

*Frontend*

```
npm install
npm run dev
```

*Backend*

Set `.env`

```
pip install -r requirements.txt
uvicorn main:app --reload
```
